## Current Focus

- Building MTN AI: a unified platform for chat and image models with transparent privacy tiers and usage-based token billing.
- Running infrastructure for hosting, IT solutions, web products, automation, and internal tools at MTN Media Group.
- Studying computer science at [FH Aachen University of Applied Sciences](https://www.fh-aachen.de/).
- Working mostly in private company repositories, so this profile is only the public slice of my engineering work.

## MTN Media Group

MTN Media Group started in March 2020 and provides IT services made in Germany. The company works across AI products, hosting, IT solutions, web design, and event services.

What I usually care about:

- Reliable backend systems and infrastructure that can be operated in production.
- Clear privacy and cost controls, especially around AI and cloud usage.
- Practical automation for support, billing, monitoring, deployment, and internal workflows.
- Interfaces that are simple enough for customers and robust enough for operations.
